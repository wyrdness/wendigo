## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with cultural sensitivity emphasis
- api.json following Wyrdness schema v1.0.0 with cultural context
- SOURCES.md bibliography prioritizing Algonquian and Indigenous authors
- Issue templates with cultural sensitivity guidelines
- Pull request template

### Data Included
- Phenomenon metadata and classification (ENTITY_SPIRIT/Indigenous Tradition)
- Cultural sensitivity and respect guidelines
- Traditional Algonquian spiritual teachings
- Geographic distribution (Algonquian territories, Northern USA/Canada)
- Historical context including starvation and survival
- Wendigo psychosis controversy documentation
- Appropriation concerns (critical level)
- Modern Indigenous interpretations
- Pop culture critique and "deer skull" imagery correction
- Resources for Algonquian cultural consultation

### Cultural Sensitivity Features
- Sacred subject matter warnings
- Historical trauma context
- Colonial diagnosis controversy addressed
- Appropriation harm documentation
- Ethical guidelines for creators/educators
- Prioritization of Indigenous voices
- Living tradition acknowledgment
